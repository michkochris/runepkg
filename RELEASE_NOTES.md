## üö¶ **Getting Started**

### **Quick Installation**
```bash
# Clone the repository
git clone https://github.com/your-username/runepkg.git
cd runepkg

# Basic build
make

# Build with Rust FFI (recommended)
make with-rust

# Run comprehensive tests
make test-comprehensive

# Install system-wide
sudo make install
```
